Push to Talk Audio Editor
^^^^^^^^^^^^^^^^^^^^^^^^^

Push to Talk allows you to edit audio in the same way you edit text.

To install on debian run:

1. `Install ipfs <https://ipfs.io/docs/install/>`_

2. ::

   $ su
