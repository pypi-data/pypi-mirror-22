As done with other Language sites, should not we generally use the plural for tags, especially when they are about grammar categories?

  • verbs
  • nouns
  • pronouns

  • tags

It is all about shared behaviour. If a behaviour is shared by all verbs, the tag should encompass all the verbs. That is why we use the plural. "Verb" only makes sense if the verb has a specific behaviour that other verbs don't, but in this case a more specific name should be prefered.
