===============================
 Invenio-PIDRelations v1.0.0a3
===============================

Invenio-PIDRelations v1.0.0a3 was released on May 18, 2017.

About
-----

Invenio module that adds PID relations to the Invenio-PIDStore module.

*This is an experimental developer preview release.*

What's new
----------

- Initial public release.

Installation
------------

   $ pip install invenio-pidrelations==1.0.0a3

Documentation
-------------

   https://invenio-pidrelations.readthedocs.io/
